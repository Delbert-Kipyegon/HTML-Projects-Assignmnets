1.0.Purpose of the Course

The purpose of the course is to provide an introduction to the Internet and the World Wide Web.

2.0. Expected Learning Outcomes

Students will be expected to:

  • Describe the structure of the World Wide Web, protocols and technologies used in web services and distributed Web applications, Web standards and standards bodies.
  • Create and validate HTML/XHTML documents, use some presentation technologies, such as Cascading Style Sheets and DHTML.
  • Describe the use of server-side backend databases in web sites and web applications.
